# 📊 National Air Quality Data Analysis & Interactive Dashboard

## 📌 Project Overview

This project presents an end-to-end analysis of the National Air Quality dataset using Microsoft Excel. The objective is to transform raw air quality data into meaningful insights through data cleaning, exploratory data analysis (EDA), KPI reporting, and an interactive dashboard.

The project demonstrates practical Excel skills used in real-world data analytics workflows, including data preparation, Pivot Table analysis, data visualization, and dashboard development.

---

## 🎯 Objectives

- Clean and prepare raw air quality data.
- Analyze AQI trends across different dimensions.
- Build KPI metrics for quick performance monitoring.
- Create interactive Pivot Tables and Pivot Charts.
- Design a professional dashboard for easy decision-making.
